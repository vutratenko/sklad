# ADR-003: Offline-first модель клиента

## Status

Accepted

## Context

PWA должна работать без сети: пользователь оперирует запасами в кладовке/на кухне с нестабильным интернетом.

## Decision

- Локальная модель данных в **IndexedDB** — источник для UI.
- **Service Worker** кэширует app shell и статику, не бизнес-данные.
- Каждое изменение создаёт запись в **op_queue** для последующей синхронизации.
- UI обновляется optimistically сразу после локальной записи.

## Consequences

**Плюсы:** работа без сети, быстрый UI, устойчивость к обрывам.

**Минусы:** сложность клиентского состояния и отладки sync.

## Rejected Alternatives

- Online-only SPA
- localStorage вместо IndexedDB
- Тонкий клиент без локального журнала операций
